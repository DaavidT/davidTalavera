## Easy caching for modern apps

If your API starts feeling slow, [Upstash](https://upstash.com/) is a quick way to add caching without managing your own Redis server.

It is a good fit for serverless projects where you want better performance with minimal ops overhead.

<InfoAlert title="Why it helps">
  Upstash gives you a fast caching layer with a lightweight setup.
</InfoAlert>

Try it here: [https://upstash.com/](https://upstash.com/)
